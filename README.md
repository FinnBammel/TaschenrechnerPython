# TaschenrechnerPython
Taschenrechner.py is an calculator where you can solve your math problems. The calculator has an grid with the numbers 0-9. It also has the feature to caculate with the operators "Addition", "subtracton", "division" and "multiplication", you do that with the buttons ont the right side or with the buttns at the bottom.
The big field at the top, is the entry field where you will see the numbers you typed in. The calculator has the function the remember the first number, as soon as you touch the operation buton like **"+", "-", "/" or "*"** the first number you typed in will be delete. After you pressed one of the operation buttons you can type in the second number. Then you press on the result button to get your total result. If you accidently typed in a wrong number or want to type in an new first number there is the function to clear the entry field with the "clear" button at the top right corner of the grid. The calculator has also the function, that you can't divide with 0. Otherwise you will see an error in the entry field.

**Functions:**
-
* Add
* Multiplicate
* Divide
* Subtract
* Clear the entry field
* Error after dividing with 0
* Number buttons from 0 to 9
